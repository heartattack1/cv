# Robert Sadretdinov — GitHub Pages

Персональная страница-визитка на GitHub Pages.

## Локальный запуск
Откройте `index.html` в браузере или используйте любой простой static-server.

## Публикация
1. Commit & push в `main`.
2. GitHub → Settings → Pages → Deploy from a branch → `main` / `/root`.
3. Откройте выданный GitHub Pages URL.

## Обновления
- Добавьте ссылки на GitHub/профили в `index.html`.
- При необходимости добавьте раздел “Selected Projects” с ссылками на репозитории.
